# COP17 Side Event 299 website

This is a static, one-page website ready for GitHub Pages.

## Publish
1. Create a public GitHub repository.
2. Upload `index.html`, `styles.css`, and the `assets` folder to the repository root.
3. Go to **Settings → Pages**.
4. Choose **Deploy from a branch**, then select `main` and `/ (root)`.
