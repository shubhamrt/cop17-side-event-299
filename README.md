# COP17 Side Event 299 — single-screen creative website

This version is designed as one immersive landing page rather than a conventional scrolling site.

## Publish on GitHub Pages
1. Create a public GitHub repository.
2. Upload `index.html`, `styles.css`, and the complete `assets` folder to the repository root.
3. Go to **Settings → Pages**.
4. Choose **Deploy from a branch**, then select `main` and `/ (root)`.
